# 🧠 AI Practicals using Python

This repository contains a series of practical implementations covering core concepts in Artificial Intelligence, Machine Learning, and Deep Learning using Python. Each practical is designed to provide hands-on experience with popular libraries and techniques.

---

## 📚 Practicals Overview

### 🔹 Practical 1: OOP, Exception Handling, and File Operations

* Implementation of Object-Oriented Programming concepts
* Handling runtime errors using exceptions
* Reading and writing files in Python

### 🔹 Practical 2: Data Cleaning, Transformation, and Analysis

* Data preprocessing using Pandas & NumPy
* Handling missing values
* Data transformation and exploratory analysis

### 🔹 Practical 3: Evaluation of ML Models

* Classification and regression model evaluation
* Metrics such as accuracy, precision, recall, F1-score, RMSE

### 🔹 Practical 4: Clustering Techniques

* K-Means clustering
* DBSCAN clustering
* Visualization and comparison of clustering results

### 🔹 Practical 5: Ensemble Learning Models

* Bagging and Boosting techniques
* Random Forest and Gradient Boosting models

### 🔹 Practical 6: Feature Engineering

* Feature scaling (Normalization, Standardization)
* Encoding categorical variables
* Feature selection techniques

### 🔹 Practical 7: Text Classification using RNN

* Natural Language Processing basics
* Building RNN models for text classification

### 🔹 Practical 8: Image Classification using CNN

* Convolutional Neural Networks
* Image preprocessing and classification tasks

### 🔹 Practical 9: Simple Deep Learning Model

* Building and training a neural network
* Model evaluation and optimization

### 🔹 Practical 10: ML Model Deployment using Flask

* Creating REST APIs using Flask
* Deploying trained ML models for real-world use

## 🚀 How to Run

1. Clone the repository

   ```bash
   git clone <your-repo-link>
   ```

2. Navigate to the project folder

   ```bash
   cd <project-folder>
   ```

3. Install dependencies

   ```bash
   pip install -r requirements.txt
   ```

4. Open Jupyter Notebook

   ```bash
   jupyter notebook
   ```
